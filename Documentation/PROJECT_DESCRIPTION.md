# Product Inventory Management System

## Objective

To develop a Product Inventory Management System using SAP RAP.

## Features

- Create Product
- Update Product
- Delete Product
- Quantity Validation
- Price Validation
- Stock Value Calculation
- OData Service Exposure
- SAP Fiori Elements UI

## Technologies Used

- SAP ABAP
- RAP
- SAP S/4HANA
- CDS Views
- OData V4
- SAP Fiori Elements
